## Stylesheets and Selectors

How do we add CSS to our HTML page?

- Inline
  <h1 style="color:blue;">A Blue Heading</h1>
- Internal
  <style>
    body {
      background-color: black;
    }

    h1 {
      color: white;
      margin: 20px;
    }
  </style>
- External
  <link rel="stylesheet" href="styles.css">
